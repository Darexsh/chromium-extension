# Google chrome Extension
Installation instructions:
- Open Google chrome (also works with Firefox Nightly (only Dev!))
- Go to `Settings` -> `Extensions` and activate `Developer mode` on the top right corner
- After activating `Developer mode`, hit `Load unpacked`
- Select the whole directory and the extension is automatically enabled

## Functions
- Center top: Time, Date and Calendar week
- Left top corner: Calendar
- Right top corner: Weather (standard location can be changed in the `script.js` file with the line `weather.fetchWeather("Deggendorf");`)
- Left bottom corner: Speedtest
- Right bottom corner: Change background picture
